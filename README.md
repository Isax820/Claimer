Description: Claimer is a lightweight and powerful plugin for securing player builds on Spigot or Paper servers. It allows players to claim chunks, manage trusted members, and automatically protect their areas from griefing.

Key Features:

Easily claim and unclaim chunks

Manage members in a claim (/claim add <player> / /claim remove <player>)

Configurable maximum claims per player

Automatic protection against block breaking and placing by non-members

/info command to view plugin version, author, and description

Saves data in config.yml for simple management

Compatibility:

Spigot & Paper (1.19+ recommended, compatible with 1.21.x)

Why choose Claimer?

Easy to install and configure

Perfect for survival, semi-RP, or community servers

Lightweight and efficient without requiring a database
